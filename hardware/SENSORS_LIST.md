# AURA Sensor Suite


| Sensor Type | Specification | Purpose |
| :--- | :--- | :--- |
| **LiDAR** | Ouster OS0 (Ultra-Wide View) | Path planning & Floor obstacle detection. |
| **Depth Camera** | Intel RealSense D435i | Gesture recognition & Face/Sign language tracking. |
| **IMU** | Bosch BNO055 | Real-time balance and orientation. |
| **Tactile Skin** | Custom Piezoresistive Matrix | Sensing patient grip strength and pain points. |
| **EMG Sensors** | MyoWare 2.0 (Integrated) | Detecting muscle intent in stroke-affected limbs. |
| **Microphone Array** | 4-Mic Circular Array | Voice command localization (for verbal patients). |
